## Description
The script that I used creates a folder with two files in it. I made a poem for the first and a mantra for the second file. The information I included was on the type of files in the script, the date, time stamp and a done message included.

## Commands used:
- date - shows the date when the script is run
- whoami - identifies the user of the script
- pwd - your location
- mkdir - creates a folder
- echo - prints the text
- cat - prints the specific file
- wc -l - counts the lines inside a file
- ls -la - list of files including hidden files
